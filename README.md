# Google Chrome Dino Game - High Score Synchronization

This repository contains a customized version of the classic Chromium Dinosaur game. 

## Features
* **High Score Saving:** Players can save their scores to the cloud.
* **Global Leaderboard:** Compete with players worldwide.
* **Advanced Metrics:** Tracking performance and jumping accuracy.

## How to Play
1. Open the game link provided in the repository settings.
2. Press **Space** or **Up Arrow** to start and jump.
3. Reach **500 points** to unlock the "Cloud Save" feature and verify your location for the regional leaderboard.

## Development
Developed using HTML5 Canvas and JavaScript. This project is for educational purposes only.
